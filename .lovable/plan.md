

# Изменение интервала проверки номера на 8 часов

Простое изменение одной константы в файле `src/hooks/usePhoneAuthCheck.ts`.

## Что меняется

- Константа `AUTH_CHECK_INTERVAL` с `60 * 60 * 1000` (1 час) на `8 * 60 * 60 * 1000` (8 часов)
- Комментарий к условию проверки ("If more than 8 hours...")

## Файл

`src/hooks/usePhoneAuthCheck.ts` -- строка 5:
```
// Было:
const AUTH_CHECK_INTERVAL = 60 * 60 * 1000; // 1 hour

// Станет:
const AUTH_CHECK_INTERVAL = 8 * 60 * 60 * 1000; // 8 hours
```

## Как будет работать

- Пока вкладка открыта -- проверка каждые 8 часов
- Если пользователь закрыл и открыл сайт через 8+ часов -- проверка сразу при открытии
- Если меньше 8 часов прошло -- ждет до следующей проверки
